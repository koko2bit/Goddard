# `@goddard-ai/cmd`

The Goddard CLI (`goddard`) is the primary developer tool for interacting with the Goddard platform. It provides commands for managing pull requests, creating feature proposals, running the local autonomous loop, and interacting with the backend.

## Issues & Feature Requests

Please direct bug reports and feature requests to the [Issue Tracker](https://github.com/goddard-ai/cmd/issues).
