You have access to a command-line tool called `goddard`. You can use it to report your status.

Commands available:

- `goddard declare-initiative --title "..."` - Use this to declare your current focus or plan.
- `goddard report-blocker --reason "..."` - Use this when you are stuck and need human assistance.
- `goddard report-completed` - Use this when you have completed the current initiative.

Use these commands by executing them in your terminal.
