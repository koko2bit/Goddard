# @goddard-ai/loop

This package contains the core logic for Goddard's agentic loop, defining how an autonomous agent is orchestrated, executed, retried, and rate-limited.

**Future Planning:**
This package is public and built to act as an entry point for running a remote agentic loop (e.g. running via Docker, systemd, or remote cloud compute) natively, without relying heavily on the outer framework.
